1. Privacy

Password: Users' passwords are encrypted using non-reversible hash encryption before entering database, so users' password security is protected.
Other Personal Information: We don't collect any other information except for user inputed info. Also, Username/Nickname is default setting to 'Anonymous'.
Aside from Username/Nickname, School Name, Gender/Sex and Major/Department are all default to none(''). Therefore if you don't want to enter those information, that's totally fine.
