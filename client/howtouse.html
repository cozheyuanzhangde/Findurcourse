How to use findurcourse.com?

How to search for courses?

1. Ignore cases.
2. Use full School Name including Campus Name like umass amherst.
3. Try to use Abbreviation for Course Subject like CS(for Computer Science). Mostly, Use the one which formatted on your school course registration website.
4. If you want to list all the courses of one school, just only type School Name and click search.
5. If you want to list all the courses of a department/subject of one school, just only type School Name and Course Subject and click search.
6. You can't list courses without specifing a School Name. Also you can't list courses with only School Name and Course Number. This feature is also prohibited. Sorry for any inconvenience.
7. You can rank the courses listed by levels of difficulty, time consumption, overall

How to Add a course?

1. Please Make Sure the course you want to add is not listed already. Sorry for any inconvenience.
2. Use mixed cases like Umass Amherst for better formatting.
3. Use full School Name including Campus Name like umass amherst.
4. Try to use Abbreviation for Course Subject like CS(for Computer Science). Mostly, Use the one which formatted on your school course registration website.
5. Enter all the fields including a comment to add a new course.

How to Add a comment?

1. Search for courses and click the course you want to check and you will be directed to coursedetail page which contains text comments of other students. Of course, you can add a comment to a existed course.

About Signup and Login

1. You can change you userinfo anytime after registration.
2. After logging in, you can comment a course or add a new course. More features will be available for members of findurcourse.com
3. Your password will be non-reversible encrypted before inserting into the database so your password is safe! Also, after you signed up with your email, you will be given a "Anonymous" username to comment on a course. You can change the username by goto user profile.
